# Netacademia Bootstrap - tegyük negyedikbe tanfolyam

https://netacademia.hu/Tanfolyam/2017bootstrap4-bootstrap---tegyuk-negyedikbe

Aki egy kicsit is járatos a webfejlesztés világában, annak nem kell bemutatni a Bootstrap -et. Ez a szupergyorsan elkészíthetõ és szép felületek világa. Nem csak a negyedik kiadással foglalkozunk majd, ha kezdõ vagy akkor is közöttünk a helyed! Nagy vonalakban: Projekt alapon fogjuk elsajátítani a webfejlesztés alapjait, a HTML, CSS alapjait. Megismerjük a Bootstrap keretrendszert és átvesszük a négyes kiadás fõbb újdonságait. Egy olyan rendszer alapjait tesszük le, ahol jegyeket lehet vásárolni, csereberélni a legjobb bulikra. Ha jól sikerül, a végén ki is adjuk :)
